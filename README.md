The Q Table file acts as a database that an AI has hopefully found to be the best move in certain game states

The EVO files are different versions of the code that ran the Machine Learning. EVO1.3 is best, though you may need to adjust the parameters of how many agents/cores are used, as well as the size of LRU_CACHE.

The PlayGoFish file allows you to play against the ML. Make sure the file name of your Q table is the same as what its reffered to in the code.
